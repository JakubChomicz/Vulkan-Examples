# Description
</div>
  
  This example shows how can you implement simple pbr in Vulkan.
  
  Sample has such features as:
  - PBR materials
  - EnvironmentMap
  - IBL
  - FXAA

  Controls:
  -
  - W, S - Zoom In Out
  - Mouse - Rotate the camera
